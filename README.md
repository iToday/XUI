# XUI
Package miniGUI interface and controls

基于MiniGUI 接口封装了Button, TextView, ImageView, Progressbar, RadioButton, ListView, PageBase， UIManager,ResourceManager;

因MiniGUI的接口和机制与Windows十分相似，此封装也可直接移植到WINCE系统中；
